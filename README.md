# charlesBerard

Portfolio Charles Berard — direction graphique.

## Stack

Next.js App Router, Sanity CMS, Tailwind v4, GSAP, Portable Text.

## Setup

```bash
pnpm install
cp .env.example .env.local
# Remplir NEXT_PUBLIC_SANITY_PROJECT_ID et SANITY_API_TOKEN
pnpm seed
pnpm dev
```

Studio Sanity : [http://localhost:3000/studio](http://localhost:3000/studio)
